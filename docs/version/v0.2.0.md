# Release Notes - v0.2.0

## 개요
v0.2.0 버전에서는 멤버 관리 시스템의 유연성을 확보하고, 보안 강화 및 실사용 편의를 위한 다양한 기능 개선이 이루어졌습니다.

## 주요 변경 사항

### 1. 멤버 관리 및 필터링 고도화
- **멤버 상태 관리**: 멤버의 활동/졸업 여부를 설정할 수 있는 필드가 추가되었습니다.
- **졸업일 기준 필터링**: 졸업한 멤버는 기본 스케줄표에서 제외되지만, 과거 스케줄 기록에는 유지되도록 로직을 개선하였습니다.
- **커스텀 다시보기 텍스트**: 멤버별로 고유한 '다시보기' 텍스트를 설정할 수 있습니다. (예: 체리 - '다시보고팡', 엘라 - '다시봐라')

### 2. 새로운 링크 및 데이터 연동
- **씨미(Cime) 전용 링크**: 멤버 프로필 내 씨미 전용 링크 필드를 추가하여 직접 연결이 가능합니다.
- **데이터 안정화**: 입력 필드에서 발생하던 데이터 반영 오류 및 `graduation_date` 미기입 시의 예외 처리를 완료하였습니다.

### 3. 보안 및 아키텍처 개선
- **RLS(Row Level Security) 정책 강화**: Supabase 보안 취약점을 해결하고 API 권한 제어를 통해 무분별한 데이터 변경을 방지합니다.
- **코드 리팩토링**: `ScheduleGrid` 및 하드코딩된 데이터들을 정리하고 DB 연동 구조로 전환하여 유지보수성을 높였습니다.

### 4. 문서 구조화
- 사용자 가이드 및 릴리즈 문서를 `docs/` 폴더 하위로 관리 체계를 통합하였습니다.

## 향후 계획
- 네이버 팬카페 링크 연동
- 홈 화면 대시보드 및 통계 기능 추가
- 다크 모드 심미성 고도화
