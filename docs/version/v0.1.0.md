# Version 0.1.0 (2026-03-14)

하나비 주간 스케줄표의 초기 안정화 및 멤버 상태 관리 기능이 구현된 버전입니다.

## 주요 변경 사항

### 1. 멤버 상태 관리 (활동/졸업)
- **데이터베이스 확장**: `characters` 테이블에 `status`, `graduation_date` 컬럼 추가.
- **졸업 멤버 처리**: 졸업일이 지난 멤버는 현재 스케줄표에서 자동으로 숨겨지며, 과거 스케줄에서는 기록이 유지됩니다.
- **관리자 UI**: 멤버 정보 수정 모달에서 활동 상태 및 졸업일을 설정할 수 있는 기능 추가.

### 2. UI/UX 개선
- **필터링 시스템**: 특정 멤버만 선택해서 볼 수 있는 필터 패널 개선.
- **모바일 최적화**: 모바일 환경에서의 가독성 향상 및 햅틱 피드백 적용.
- **이미지 내보내기**: 현재 스케줄표를 PNG 이미지로 저장하는 기능 고도화.

### 3. 버그 수정 및 기타
- Push Notification 관련 간헐적 오류 해결.
- 멤버별 컬러 테마 및 아바타 이미지 처리 로직 안정화.
- 씨미(Cime) 아이콘 및 링크 교체.

## 향후 로직 예정
- 네이버 팬카페 링크 연동.
- 스케줄 그리드 컴포넌트 리팩토링 및 성능 최적화.
