# 하나비 스케줄표 사용 가이드

이 문서는 하나비 주간 스케줄표를 이용하는 일반 사용자와 관리자를 위한 가이드입니다.

## 1. 일반 사용자 가이드

### 스케줄 확인하기
- 메인 화면에서 이번 주의 통합 스케줄을 한눈에 확인할 수 있습니다.
- 상단 중앙의 날짜 범위를 클릭하면 다른 주차의 스케줄로 이동할 수 있습니다.

### 특정 멤버만 보기 (필터)
- 우측 상단의 **[메뉴]** 버튼 또는 **[필터]** 버튼을 클릭하여 보고 싶은 멤버만 선택할 수 있습니다.
- "전체 선택" 또는 "해제" 버튼을 통해 빠르게 필터를 설정할 수 있습니다.

### 이미지로 저장 및 공유
- **[이미지로 저장]** 버튼을 누르면 현재 보고 있는 스케줄표가 PNG 이미지로 생성됩니다.
- 모바일 기기에서는 바로 공유하기 창이 뜨며, PC에서는 파일로 다운로드됩니다.

### 구글 캘린더 등에 추가
- **[캘린더 추가 (ICS)]** 버튼을 통해 본인의 일정 관리 앱(구글 캘린더, 애플 캘린더 등)에 하나비 멤버들의 스케줄을 가져올 수 있습니다.

---

## 2. 관리자 가이드

### 스케줄 편집
- `/admin` 경로를 통해 관리자 모드에 진입하면 스케줄 내용을 직접 수정할 수 있습니다.
- 각 셀의 내용을 클릭하여 시간과 컨텐츠를 입력하고, 방송/Off/합방 등의 타입을 설정하세요.

### 멤버 관리 (졸업 설정 등)
- 관리자 화면 사이드바의 멤버 목록에서 ✏️ 아이콘을 누르면 멤버 정보를 수정할 수 있습니다.
- 멤버가 졸업하는 경우, 상태를 **'졸업'**으로 변경하고 **졸업일**을 입력하면 해당 일자 이후부터 스케줄표에서 자동으로 제외됩니다.
