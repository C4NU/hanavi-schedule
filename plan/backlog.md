## v1.6.0+ (예정)
- [ ] **DB 자동 백업 시스템**: 안정적인 데이터 보관을 위한 자동화된 백업 프로세스 구축.
- [ ] **일정 변동 코멘트 기능**: 멤버의 라이브 언급 등을 기반으로 특정 일정에 코멘트를 남길 수 있는 기능 (반짝이 기입용).
- [ ] **주간 테마 뷰 (Everytime Style)**: 에브리타임 감성의 1주일치 전체 시간표 뷰 제공.
- [ ] **일요일 단체 합방 레이아웃**: 일요일 정기 합방 시 스케줄 영역을 하나로 통합하여 가시성 개선.
- [ ] **아침 스케줄 요약 푸시**: 매일 아침 당일 방송 스케줄 요약 정보를 알림으로 제공.
